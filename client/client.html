Call of Duty Zombie Map Loader

This form is going to be used to load a JSON file containing data on maps from the Call of Duty Zombie series. The file lists off multiple maps from throughout the games and various aspects about them. I will attempt to do the following with this:

  • Parse the file and break down each map based on what the user wants to look for.
  • The search is done with a dropdown menu that will look through a designated component (name, enemy types, etc.)
  • I will attempt to carry over the buttons from my other API projects which will display one search at a time and the user can scroll through them with a 'prev' and 'next' button.

This is how I will use the status codes:

  • 200: Retieve any saved maps added to the array
  • 201: Called when the map is added to the array when it is first searched
  • 204: Called when the map selected is already added to the array
  • 400: Called when the search paramter is empty
  • 404: Called when the server can't return what was searched
  • 500 (if necessary): If the server encounters a situation it doesn't know how to handle

Try and add a way for the user to add data into the saved data
